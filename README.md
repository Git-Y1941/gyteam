# 测试版v-0.1（2026-3-10）
## 固原支教队前端项目
## 项目介绍

本项目是厦门大学西部梦想社团固原支教队的官方网站前端部分，旨在展示支教队的活动、成员信息、资源分享等内容，为队员和关注支教事业的人士提供一个交流和了解的平台。

## 技术栈

- HTML5
- CSS3
- JavaScript

## 功能说明

### 1. 首页
- 轮播图展示支教活动照片
- 置顶通知
- 功能按钮导航
- 热门资讯展示

### 2. 队伍介绍
- 队伍历史和使命介绍
- 成员展示
- 组织结构

### 3. 当届动态
- 当届支教活动的最新动态
- 活动照片和详情

### 4. 朝花夕拾
- 往届支教活动回顾
- 历史照片和故事

### 5. 资源分享
- 教案模板和范例
- 活动策划案
- 预算表
- 资源筛选功能

### 6. 发现
- 资讯列表
- 分类筛选
- 圆角设计和动画效果

### 7. 个人中心
- 游客登录和队员登录
- 个人信息展示
- 队员标签显示
- 退出登录功能

### 8. 登录页面
- 队员登录和游客登录选项
- 轮播图背景

   - 建议保持代码风格一致，遵循现有命名规范
